## 手机端操作指南（预览）

### Tasker + HTTP POST 方案（推荐）

1. **安装 Tasker**（Google Play，付费 ¥25）
2. **安装 AutoApps 插件**（免费，获取屏幕使用时间）
3. **配置 Tasker 任务:**
   - 每 30 分钟触发
   - 读取 UsageStatsManager 获取 App 使用时间
   - HTTP POST 到服务器 `/api/phone/stats`
4. **数据格式:**
```json
{
  "device_id": "android_xxx",
  "timestamp": "2026-07-05T22:00:00",
  "screen_time_total": 18000,
  "app_usages": [
    {"package": "com.ss.android.ugc.aweme", "name": "抖音", "duration": 3600},
    {"package": "com.tencent.mm", "name": "微信", "duration": 1800}
  ]
}
```

### 备选方案：Termux + 脚本

1. **安装 Termux**（F-Droid 版本）
2. **安装 Termux:API**（获取 UsageStats）
3. **运行 Python 脚本定时上报**
